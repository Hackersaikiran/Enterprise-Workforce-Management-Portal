# Enterprise Workforce Management Portal

A modern, responsive Enterprise Workforce Management Portal built with semantic HTML5, modern CSS3, and minimal JavaScript for theme persistence.

![Enterprise Workforce Management Portal Preview](assets/preview.png)

## Features

- Multi-page enterprise dashboard (Dashboard, Employees, Projects, Calendar, Reports, Settings)
- Responsive layout optimized for desktop and mobile
- Light/Dark mode with persistence across pages
- CSS-only dropdowns with click-outside close handling
- Semantic HTML5 structure and modern CSS3 styling
- Clean enterprise UI components (cards, tables, badges, modals)

## Pages

- Landing: index.html
- Dashboard: dashboard.html
- Employees: employees.html
- Projects: projects.html
- Calendar: calendar.html
- Reports: reports.html
- Settings: settings.html

## Getting Started

Open index.html in your browser or serve the folder with any static server.

## Theme

The theme toggle is persisted using localStorage via theme.js. If you want to reset it, clear the browser storage for this site.
